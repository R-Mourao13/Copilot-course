# 🔧 Bolt Ranger 3D

Jogo de ação **3D** para o telemóvel (iPhone/Android) e browser, inspirado em
**Ratchet & Clank**. Feito em WebGL com Three.js, com controlos touch (joystick +
botões) e suporte a PWA (instalável no ecrã principal, funciona offline).

➡️ **O jogo está na pasta [`game/`](game/).** Vê o [`game/README.md`](game/README.md)
para instruções de como jogar, servir e instalar no iPhone.

## Arranque rápido

```bash
cd game
python3 -m http.server 8000
# abre http://localhost:8000 no browser
```
